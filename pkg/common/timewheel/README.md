# timewheel

为心跳上报定制实现的、线程安全的多层时间轮，简化功能以追求更好的性能。

基于链表和ticker实现。 只有插入操作：插入任务后就必须执行。

## 测试/压测记录

覆盖率： 90%+

1. 功能测试：正常

2. 压力测试： 16c 16g机器
