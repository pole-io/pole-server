# pole-io

基于 Tencent 开源的 [polaris](https://github.com/polarismesh/polaris) 进行的二开项目

第一个二开版本预期支持的新特性

- [ ] 兼容 consul、apollo、sofa 协议接入
- [ ] 更好的服务注册发现性能以及内存控制
- [ ] 支持服务实例的主动探测
- [ ] 更强的配置中心，支持多环境版本渲染，降低用户的配置维护理解成本
- [ ] 所有治理规则均支持版本控制以及灰度下发
- [ ] 所有版本的北极星客户端都将支持标签灰度下发能力
- [ ] 所有资源均支持细粒度鉴权，对标云厂商的 CAM/RAM 等能力
